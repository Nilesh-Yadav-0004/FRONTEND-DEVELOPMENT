# FRONTEND-DEVELOPMENT

How to Open the repository from GITHUB to  VSCODE

Step 1:- create Repository and add Readme file on. 

step 2:- Copy the URL for the repo.

step 3:- Open gitbush by right click.

step 4:- inside write git clone and paste the URL.

step 5:- Open File Manager and select the folder and write cmd on top .

step 6:- Open VSCODE and open the folder on it and then ready to go.
